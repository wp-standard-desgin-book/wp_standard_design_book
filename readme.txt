★インポートファイル(/import 内 XMLファイル）について★
---------------------------------------------------
WordPress Importerプラグインを使用し、本書サンプルサイト用のコンテンツをインポートできます（本書 APPENDIX01参照）。
インポートファイルは以下の5種類を用意しましたので、必要に応じて使用してください。
 
 all_data.xml（全コンテンツ）
 posts_data.xml（投稿のみ）
 pages_data.xml（固定ページのみ）
 nav_menus.xml （ナビゲーションメニューのみ）
 menus_data.xml（メニュー投稿タイプのみ）

ただし、コンテンツ内で使用する画像データについては上記XMLファイルからはインポートできませんので、
管理画面内、投稿編集画面から適宜アップロードし・投稿内やアイキャッチ画像に設定してください。
